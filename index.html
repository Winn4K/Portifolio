Tecnologias

Javascript

O JavaScript é uma linguagem de programação de alto nível voltada para o desenvolvimento web. Originalmente criado para funcionar nos navegadores, ele permite criar páginas interativas com elementos dinâmicos e boa performance

HTML

O HTML5 é a evolução mais recente da linguagem de marcação de hipertexto, permitindo a criação de sites modernos e compatíveis com qualquer navegador. Ele expande o HTML clássico com novas tags e recursos

CSS

CSS é uma linguagem de estilo usada para controlar a apresentação visual de documentos HTML. Enquanto o HTML define a estrutura e o conteúdo de uma página web, o CSS determina como esse conteúdo é estilizado e exibido aos usuários

React

O React é uma biblioteca JavaScript usada para criar interfaces de usuário (UI) em aplicativos web. Ele permite dividir uma página em partes chamadas de componentes, facilitando o desenvolvimento e a manutenção do código

Bootstrap

Bootstrap é um framework front-end de código aberto que combina CSS e JavaScript para estilizar elementos de páginas escritas em HTML. Ele fornece componentes prontos para agilizar o desenvolvimento web, permitindo criar sites e aplicações responsivas

SASS

SASS (Syntactically Awesome Style Sheets) é um pré-processador de CSS que permite escrever estilos com uma sintaxe mais intuitiva e adicionar recursos não presentes no CSS puro

Java

Java é uma linguagem de programação orientada a objetos que permite os desenvolvedores escreverem programas uma vez e executa-los em diferentes dispositivos, sendo amplamente usada no desenvolvimento de aplicativos e serviços digitais
